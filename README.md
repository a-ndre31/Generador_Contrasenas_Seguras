# Generador Inteligente de Contraseñas Seguras
#  Descripción

Aplicación web desarrollada en Python con Flask que genera contraseñas seguras mediante generación aleatoria y cálculo de entropía. 

Incluye un modo accesible orientado a adultos mayores y personas con baja alfabetización digital, permitiendo generar contraseñas fáciles de recordar pero seguras.



#  Funcionalidades

-  Generación de contraseñas de alta seguridad
-  Generación de contraseñas memorables
-  Cálculo de entropía en bits
-  Evaluación de fortaleza (Débil, Media, Fuerte, Muy Fuerte)
-  Indicador visual con barra de progreso
-  Botón para copiar al portapapeles
-  Arquitectura modular (N-Tier)

# Cómo Ejecutar el Proyecto
1. Clonar el repositorio:
https://github.com/a-ndre31/Generador_Contrasenas_Seguras.git
2. Crear entorno virtual:
python -m venv venv
3. Activar entorno:
venv\Scripts\activate
4. Instalar dependencias:
pip install flask
5. Ejecutar:
python app.py
6. Abrir en navegador:
http://127.0.0.1:5000/
## 📊 Seguridad

La fortaleza de las contraseñas se evalúa mediante cálculo de entropía:

Entropía = longitud × log₂(conjunto de caracteres únicos)
